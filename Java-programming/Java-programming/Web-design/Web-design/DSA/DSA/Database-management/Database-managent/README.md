 Database Management Systems (DBMS)
This folder contains SQL scripts focusing on relational database design, data integrity, and complex queries.

Learning Roadmap

Level 1: SQL Basics
    Understanding Data Types and Table Creation.
    Practicing CRUD (Create, Read, Update, Delete) operations.

Level 2: School Management Database
    Schema Design: Creating tables for `Students`, `Umits`, and `Grades`.
    Data Population: Inserting sample student records and course details.
    Relational Queries: Using `JOIN` statements to link students to their specific courses and grades.

Level 3: Data Integrity
    Implementing Primary Keys and Foreign Keys to maintain relationships.


Objective: Design a relational system for university data management.

Feature: A 3-table schema (Students, Units, Grades) utilizing INNER JOIN queries.

Why: Uses Normalization to eliminate data redundancy and ensure database integrity.
